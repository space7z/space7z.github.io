urllib

基本使用


一个类型和六个方法

urllib.request

一个类型：

  • HTTPResponse类型

六个方法：

  • read()：一行一行读
  • read(n)：读取n个字节
  • readline()：一次读取一行
  • readlines()：一次读取一行，读完为止
  • getcode()：获取响应状态码
  • geturl()：获取url地址
  • getheaders()：获取一个状态信息

get请求的uquote方法

urllib.parse.quote(string)：

  • 将字符串转为unicode编码

get请求的urlencode方法

urllib.parse.urlencode()：

  • 将字典类型的请求参数转为unicode编码类型地址

post请求方式
